CSS (Cascading Style Sheets)

FiturFungsi / Penjelasan LengkapContoh / Cara Pakai
SelectorDigunakan untuk memilih elemen HTML tertentu agar bisa diberikan style. Misalnya memilih semua paragraf, div tertentu, atau elemen dengan class/id tertentu.p { color: red; } .box { margin: 10px; } #header { background: blue; }
Box ModelMengatur ruang elemen meliputi content, padding, border, dan margin. Penting untuk memahami layout dan jarak antar elemen.div { padding: 10px; border: 2px solid black; margin: 5px; }
DisplayKontrol bagaimana elemen ditampilkan: block, inline, flex, grid, atau hidden. Sangat penting untuk layouting.div { display: flex; } span { display: inline; }
PositionMenentukan posisi elemen dalam halaman: relative, absolute, fixed, sticky.div { position: absolute; top: 10px; left: 20px; }
FlexboxLayout fleksibel satu dimensi, mengatur arah (row/column), alignment, dan wrapping elemen anak.div { display: flex; justify-content: center; align-items: center; }
GridLayout dua dimensi untuk baris dan kolom, mengatur area dan jarak antar elemen.div { display: grid; grid-template-columns: 1fr 2fr; gap: 10px; }
TypographyMengatur font, ukuran, spasi antar huruf, tinggi baris, dan gaya teks untuk tampilan lebih enak dibaca.p { font-family: Arial; font-size:16px; line-height:1.5; letter-spacing:0.5px; }
ColorsMenentukan warna teks dan background. Bisa menggunakan HEX, RGB, HSL, atau variabel CSS.p { color: #ff0000; background-color: rgb(0,0,255); }
BackgroundMengatur background color, gambar, posisi, dan pengulangan.div { background-image: url('img.jpg'); background-size: cover; }
Pseudo-classMemberikan style untuk kondisi tertentu seperti hover, focus, active.a:hover { color: red; } input:focus { border-color: blue; }
Pseudo-elementMembuat elemen virtual sebelum atau sesudah konten untuk dekorasi atau styling tambahan.p::first-line { font-weight: bold; } div::after { content:'*'; }
ResponsiveMembuat layout dan elemen menyesuaikan ukuran layar menggunakan media queries.@media (max-width:600px) { body { font-size:14px; } }
TransitionMemberikan efek perubahan halus ketika properti CSS berubah.div { transition: all 0.3s ease; }
AnimationMembuat animasi kompleks dengan keyframes dan property animasi.@keyframes slide { from {opacity:0;} to {opacity:1;} }
TransformEfek transformasi elemen seperti rotasi, scaling, translasi, skew.div { transform: rotate(45deg) scale(1.2); }
VariablesMenyimpan nilai yang bisa digunakan ulang di CSS agar konsisten.:root { --main-color: red; } p { color: var(--main-color); }
Z-indexMenentukan lapisan elemen dalam stack, lebih besar lebih di atas.div { position: relative; z-index: 10; }

Tailwind CSS

KategoriFungsi / Penjelasan LengkapContoh / Cara Pakai
LayoutUtility class untuk mengatur struktur halaman, membuat container, mengatur tampilan blok, flex, atau grid.<div class='container mx-auto flex grid hidden'></div>
Flex/GridMengatur arah dan alignment elemen anak, mengatur jumlah kolom grid dan jarak.<div class='flex flex-row items-center grid grid-cols-3 gap-4'></div>
SpacingMengatur margin dan padding secara cepat tanpa menulis CSS manual.class='p-4 px-2 m-2 mt-4'
SizingMengatur ukuran lebar, tinggi, dan maksimum elemen.class='w-full w-1/2 h-screen max-w-lg'
TypographyMengatur font, ukuran, ketebalan, dan tracking teks.class='text-xl font-bold tracking-wide'
ColorsMengatur warna teks, background, border dengan skala Tailwind.class='text-red-500 bg-blue-200 border-gray-400'
Border/RadiusMengatur garis tepi dan sudut elemen dengan cepat.class='border rounded-md rounded-full'
BackgroundMengatur background color atau gradient menggunakan utility class.class='bg-gradient-to-r from-blue-500 to-green-400'
EffectsShadow, opacity, dan efek visual lain agar elemen lebih menarik.class='shadow-lg opacity-50'
TransformMengatur transformasi seperti scale, rotate, translate.class='transform scale-105 rotate-45'
TransitionEfek perubahan halus saat state berubah.class='transition-all duration-300'
Order/Grid SpanMengatur urutan elemen dalam flex/grid atau span kolom/row.class='order-1 col-span-2'
ResponsivePrefix sm:, md:, lg: untuk menyesuaikan style per breakpoint.class='sm:w-1/2 lg:grid-cols-4'
Dark ModeMengatur style untuk mode gelap menggunakan class 'dark'.class='dark:bg-gray-900 dark:text-white'
State VariantsMengatur state interaksi seperti hover, focus, active.class='hover:bg-blue-500 focus:ring'

JavaScript

FiturFungsi / Penjelasan LengkapContoh / Cara Pakai
VariabelMenyimpan data. Gunakan var, let, const untuk mendeklarasikan variabel.let name = 'Fando'; const age = 25; var isActive = true;
Tipe DataMenyimpan tipe primitif (string, number, boolean) atau reference (object, array).let str = 'Hello'; let num = 10; let arr = [1,2,3]; let obj = {name:'Fando'};
OperatorMelakukan operasi aritmatika, logika, perbandingan, dan ternary.x + y, x && y, x === y, condition ? a : b
FunctionBlok kode yang bisa dipanggil ulang, termasuk function biasa dan arrow function.function add(a,b){return a+b;} const mult = (x,y)=>x*y;
ScopeLingkup hidup variabel: global, lokal, atau block.let a=1; { let b=2; }
ObjectMenyimpan data dengan key:value.let user = {name:'Fando', age:25};
ArrayList data berurutan, bisa diiterasi dan diubah.let nums = [1,2,3]; nums.map(n=>n*2);
PrototypeMewarisi properti/metode di JavaScript.function Person(){} Person.prototype.sayHi = ()=>{};
ClassBlueprint OOP di JS untuk membuat object.class User{ constructor(name){this.name=name;} }
DOMManipulasi HTML/elemen di halaman.document.querySelector('#id').textContent = 'Hello';
EventMenangani aksi user seperti klik, submit, input.button.addEventListener('click',()=>alert('clicked'));
AsyncMenangani operasi non-blocking seperti timer atau request API.setTimeout(()=>console.log('done'),1000); fetch(url);
PromiseMenangani hasil async secara chaining.fetch(url).then(res=>res.json()).then(data=>console.log(data));
Async/AwaitSintaks modern untuk menulis async lebih mudah dibaca.const data = await fetchData();
ModuleMembagi kode menjadi file yang bisa diimport/export.import {func} from './file.js'; export function func(){};
Error HandlingMenangani error agar tidak crash.try { risky(); } catch(e){ console.log(e); }

TypeScript

FiturFungsi / Penjelasan LengkapContoh / Cara Pakai
Tipe DataMenentukan tipe data variabel agar lebih aman dan mudah dipahami.let age: number = 25; let name: string = 'Fando'; let active: boolean = true;
Array & TupleArray memiliki tipe tunggal, tuple memiliki urutan dan tipe tetap.let arr: number[] = [1,2]; let tup: [string,number] = ['A',1];
UnionVariabel bisa memiliki lebih dari satu tipe data.let val: string | number; val = 'hello'; val = 123;
IntersectionMenggabungkan beberapa tipe menjadi satu.type A = X & Y; let obj: A = {...};
InterfaceMenentukan kontrak struktur object agar konsisten.interface User { name: string; age:number; }
Type AliasMemberi nama lain untuk tipe data kompleks.type ID = string | number;
